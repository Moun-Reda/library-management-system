# Library Management System Using PHP & MySQL

This project is a web-based **Library Management System** developed using **PHP, MySQL, HTML, and CSS**.  
It allows administrators and users to manage books, members, and borrowing operations efficiently.

The system was developed to demonstrate practical skills in backend development, database management, and web application design.
